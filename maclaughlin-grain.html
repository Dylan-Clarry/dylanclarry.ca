MacLaughlin Grain

MacLaughlin Grain is a custom live edge woodworking business based in Markham Ontario, building everything from charcuterie boards to side tables and dining tables. The website maclaughlingrain.ca was developed to allow the owner to upload products on the fly and give customers a simple way to purchase products and request work.


This project is a full stack CRUD (create, read, update, delete) web-app to manage inventory for the business. It features a RESTful API built in express and a client side single page application built in vanilla JavaScript for a smooth app-like feel.

Project Links

Frontend

The frontend of the website is designed to be minimalistic and to have the products be the star of the show. Designed with Figma, the website is fully responsive and works across all devices.


Built as a single page app, the website is made to feel snappy and instantaneous to get an app-like feel and responsiveness.

Backend

The backend features a RESTful API built in Express and works asynchronously with the client side to deliver product information and to allow the business owner to create, read, update, and delete products. This API also features an emailing system for contact forms and product requests.
